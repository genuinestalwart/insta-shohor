How does javascript work?

JavaScript is client-side scripting language, mostly used for making a website interactive and dynamic as well as building a web app. Client-side in web development means, anything a user sees on their browser. On the other hand, scripting language means interpreting the code on runtime instead of compiling it. One of the reasons for its popularity is because javascript is the most light programming language and can be easily loaded on slow internet.

What is the difference between local storage and session storage?

The main difference between local storage and session storage is for how long they store data and when it gets expired. For session storage, Whenever a web page is loaded in the browser, a unique page session is created and assigned to the tab, where the page is loaded.Means, if you close the tab or the browser, the session will get expired. On the other hand, the data of local storage has no expiration time.
